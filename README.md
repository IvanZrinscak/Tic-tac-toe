 # Tic-tac-toe
 A generalization of the famous Tic-tac-toe game on a n^d field.
+
+## Web interface
+
+To play a simple 3x3 version in your browser, install Flask and run:
+
+```
+pip install Flask
+python web_app.py
+```
+
+Then open `http://localhost:5000` in your browser and play the game.
